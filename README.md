## Installation

In your Hugo site `themes` directory, run:

```
git clone https://github.com/tprzybylek/hugo-terra
```

Next, open `config.toml` in the base of the Hugo site and set the theme option to `hugo-terra`.

```
theme = "hugo-terra"
```
